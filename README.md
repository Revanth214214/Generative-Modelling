# Generative Modelling Case Study Using GANs

This repository contains the implementation, experiments, and final report for a university case study on **Generative Adversarial Networks (GANs)** across multiple data domains.

## Project Overview

The objective of this case study was to investigate the practical performance of GANs on different types of data, including synthetic datasets, medical images, cybersecurity traffic records, and sketch-based image generation.

The study includes both qualitative and quantitative evaluation using visual comparisons, training loss curves, Principal Component Analysis (PCA), and Fréchet Inception Distance (FID).

## Tasks Completed

### Part 1: GAN on Synthetic 2D Data
- Reproduced a GAN on sine-wave data
- Applied GANs to Gaussian mixture distributions
- Compared baseline and modified architectures

### Part 2.1: BloodMNIST Medical Image Generation
- Implemented DCGAN for blood cell image synthesis
- Evaluated results using FID
- Completed extension task using Conditional GAN (cGAN)

### Part 2.2: CICIDS2017 Cybersecurity Traffic Generation
- Applied GANs to tabular network traffic data
- Compared real and generated samples using PCA

### Part 2.3: QuickDraw Pizza Sketch Generation
- Implemented DCGAN for hand-drawn pizza sketch generation
- Improved FID through model fine-tuning
